## Used Technology

- TypeScript
- NextJs
- Redux
  - Redux Toolkit
- Axios
- React-quill
- React Hook form
  - Hook Form Resolvers
- Yup
- Tailwind
- Jwt-decode
- ESLint

## Main Functionality

- Registration and login
- Edit profile
- Change password
- View profile
- View all blogs with pagination and searching
- View blog details
- Create blog
- Update blog
- Delete blog
- Like on blog
- Remove like
- Comment on blog
- Delete own comment
- Replay on comment
- Delete own replay
